Welcome To Expense Tracking Dashboard



What is Expense Tracking Dashboard


This system helps companies efficiently manage employee expenses. Employees can submit expense claims with details like amount, category, description, and date.
Managers can review, approve, or reject these claims, and admins can oversee the entire process, manage users, define roles, and configure approval workflows.
The system ensures transparency, simplifies expense tracking, and provides real-time status updates for all submitted claims.







Features Of Dashboard




Employee Features


Submit expense claims with amount, category, description, and date.

View their own expense history and approval status.




Manager Features


Approve or reject expense claims with comments.

View team expenses and pending approvals.

Escalate expenses as per company rules.




Admin Features


Auto-create company and admin on signup.

Manage users, assign roles, and define manager relationships.

Configure approval rules and override approvals.

View all company expenses in a single dashboard.
